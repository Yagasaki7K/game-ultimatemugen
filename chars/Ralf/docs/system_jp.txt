-- 基本操作 --
弱パンチ				X
強パンチ				Y
弱キック				A
強キック				B
ふっとばし攻撃				C
緊急回避				X+AorZ
挑発					Start
パワーため				2+Start

-- ダッシュ --
「config.txt」で設定するか初回ラウンドのイントロ中にスタートを押すことで変更できます。

-DASH
ダッシュ				66
クイック緊急回避			(通常技ヒット中)X+AorZ

-STEP
ステップ				66
クイック緊急回避			(通常技ヒット中)X+AorZ
歩行速度が他のタイプより速くなります。

-G.STEP
グランドステップ			66
GCグランドステップ			(ガード中)66
ACグランドステップ			(通常技ヒット中)X+AorZ

-- ジャンプ --
小ジャンプ				短くレバー上
中ジャンプ				短くレバー下の後、短くレバー上
大ジャンプ				短くレバー下の後、レバー上

-- MAX発動 --
同時押しのボタンを原作仕様(Y+A)にしたい場合は「config.txt」で設定を変更してください。

MAX発動(KOF'98タイプ)			2+Y+B
MAX発動(KOF'02タイプ)			Y+B
クイックMAX発動				(通常技ヒット中)Y+B
  →クイックダッシュ			6押しっぱなし

-- ガードキャンセル --
ガードキャンセルふっとばし攻撃		(ガード中)C
ガードキャンセル緊急回避		(ガード中)X+AorZ

-- 特殊キャンセル --
スーパーキャンセル			スーパーキャンセル対応技中に超必殺技orMAX超必殺技
ドリームキャンセル			ドリームキャンセル対応技中にMAX超必殺技

-- ゲージ消費量 --
GCグランドステップ			1ゲージ
ガードキャンセルふっとばし攻撃		1ゲージ
ガードキャンセル緊急回避		1ゲージ
クイック緊急回避			1ゲージ
ACグランドステップ			1ゲージ
MAX発動					1ゲージ
クイックMAX発動				1.5ゲージ
超必殺技				(通常時)1ゲージ   / (MAX発動時)MAX発動終了
EX超必殺技				(通常時)1.5ゲージ / (MAX発動時)0.5ゲージ+MAX発動終了
MAX超必殺技				(通常時)2ゲージ   / (MAX発動時)1ゲージ+MAX発動終了
MAX2					(通常時)3ゲージ   / (MAX発動時)2ゲージ+MAX発動終了
